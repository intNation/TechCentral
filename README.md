# TechCentral

TechCentral is a C# ASP.NET e-commerce web application for browsing, purchasing, and managing technology products.  
The system includes customer shopping functionality, checkout and invoice generation, administrative product management, and reporting features.

The project is structured using a web layer and a service layer, with a local SQL Server database.

---

## Purpose

This project was developed as a group software development project to implement a basic e-commerce system using ASP.NET and C#.

The focus was on:
- multi-page web applications
- database integration
- service-layer architecture
- user account management
- checkout and invoice workflows
- admin product management

---

## Features

### Customer Features
- Browse technology products
- View products by category
- View individual product details
- Add products to cart
- Wishlist functionality
- Checkout process
- Invoice generation
- View previous invoices
- User registration and login

---

### Admin Features
- Add products
- Edit products
- Delete products
- Product management dashboard
- Sales / data reporting

---

### System Features
- Authentication (login / signup / logout)
- Shopping cart management
- Invoice management
- Product catalog system
- Category filtering
- Basic analytics/reporting
- Database-driven application

---

### Web Layer
Handles:
- UI rendering
- customer interaction
- admin pages
- checkout workflow

Built using:
- ASP.NET Razor Pages
- C#

---

### Service Layer
Handles:
- business logic
- product operations
- database access
- invoice processing

## Running the Project

### Requirements
- Visual Studio 2019 or later
- .NET SDK
- SQL Server LocalDB

### Steps
1. Open the solution in Visual Studio.
2. Build the project.
3. Run the Service project.
4. Run the Web project.
